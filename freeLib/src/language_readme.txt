Исправление ошибок в переводе
1. Установите программу Linguist (http:\\http://qt-project.org/)
2. Откройте в ней файл перевода, который вы хотите исправить (language_*.ts)
3. Откорректируйте перевод.
4. Выполните команду Файл->Скомпилировать
5. Получившийся файл с расширением qm (language_*.qm) запишите в папку language, находящеюся в рядом с исполняемым файлом программы
    MacOS X - freeLib.app\Contents\MacOS\language
    Linux   - freeLib\bin\language
    Windows - freeLib\bin\language
6. Перезапустите программу
    
Создание нового перевода
1. Установите программу Linguist (http:\\http://qt-project.org/)
2. Выберите файл перевода, на основании которого вы будете делать новый перевод, например: language_en.ts
3. Создайте копию этого файла заменив en на выбранный язык
4. Откройте в программе Linguist созданный файл
3. Откорректируйте перевод.
4. Выполните команду Файл->Скомпилировать
5. Получившийся файл запишите в папку language, находящеюся в рядом с исполняемым файлом программы
    MacOS X - freeLib.app\Contents\MacOS\language
    Linux   - freeLib\bin\language
    Windows - freeLib\bin\language
6. Перезапустите программу
